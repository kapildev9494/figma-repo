# Grids & Spacing

Delish component library provides a flexible and responsive grid system that enables you to create well-structured and visually appealing layouts. The grid system allows you to easily arrange and align your content.

Our library offers a comprehensive set of spacing utilities to control the spacing between elements. These utilities provide consistent and predictable spacing throughout your application.
